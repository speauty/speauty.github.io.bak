# 者也

> “之乎者也助得甚事”    --北宋·文莹《湘山野录》

颇有趣味，如今看来，虽是讥讽之意。不过也可从中警戒自己，不能一味掉书袋，毕竟实践出真知，也不是没有道理的。

简而言之，实践是必需的，经验是总结出来的。为此，念头通达后，就打算重新搞下博客。有些东西还是需要交流和碰撞，该得瑟就得瑟，该服就得服。人啊，就不能太扭捏。一个字，难看。

者也，者也，之乎者也。切记，人生，不只有工作，还有生活，不要在茫茫人海中迷失了自我。

贫道去去就回，勿念。有兴趣的话，可以看看这个。
